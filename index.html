insideViewport.js

An ultra light weight library that detects if an element is in viewport and lets you do some action to it.

Docs GitHub

Why Use it?

  • An ultra light weight (JUST 2.2KB).
  • Written in pure vanilla javascript.
  • Compatible with all browsers.
  • Compatible with even old browsers versions, not only modern ones.
  • No special setup or npm package install needed.
  • Just copy and paste less than 30 lines of code at the top of your JS file.
