syntax

» Call insideViewport( { } ) with an object or more objects.

» First property: classesNames an array of strings of classes you want to detect.

» Second property(optional): position a number value from 0 to < 1. It controls position of element in viewport when the action done on it.

» Third property: action(elements) which is a function to do some action on the element when in viewport.

                                
insideViewport({
  classesNames: ['example1', 'example2'],
  position: 0.5,
  action: function(elements) {
    myFunction(elements)
  }
},
{
  classesNames: ['example3', 'example4'],
  action: function(elements) {
    myOtherFunction(elements) 
  }
})
